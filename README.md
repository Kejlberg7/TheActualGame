# TheActualGame

A Java 24 game project.

## Getting Started

### Build
```
gradle build
```

### Run
```
gradle run --args=''
```

### Test
```
gradle test
```

## Project Structure

- `src/main/java` - Java source code
- `src/main/resources` - Game assets and resources
- `src/test/java` - Test code
- `src/test/resources` - Test resources
